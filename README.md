# React Webpack Testing

## Requirements

- [Node.js](http://nodejs.org): Use the installer (LTS) provided on the NodeJS website. --node: v16.13.2

## Developing

- Clone this repo
- Open terminal and `cd <path to repo>`
- run `npm install`
- run `npm start`
- browse to <a href="https://localhost:9000">localhost:9000</a>
